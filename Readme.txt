安装
-----------------------------------------------
将libsvm-3.20放到MATLAB的子目录\toolbox下。启动MATLAB并从文件菜单中选择设计路径-添加并包含子文件夹，在文件对话框中选择文件夹libsvm-3.20，点击确定，再点击保存就安装好了。


特点
-----------------------------------------------
该工具箱实现了对原核生物σ54启动子高精度的预测，主要通过SVM_Adaboost.m 程序实现，并包含：
1.k-tuples编码技术
2.参数网格寻优技术
3.特征选择技术
4.SVM-Adaboost算法


用法
-----------------------------------------------
1.将SVM-Adaboost中的K-tuples.txt打开，这是一个C++程序，用任意能打开C++程序的软件运行它并保存运行的结果，这里命名为K_tupledata.txt（为了方便大家使用，K_tupledata.txt这个文件我  也附在了SVM-Adaboost文件根目录中）；
2.将待预测的DNA序列用BLAST程序截取-20 bp到+60 bp；
3.打开libsvm-3.20\matlab（已经安装好的目录）中的K_tuples.m，修改data1和data2的路径，data1修改为待预测序列路径（经过BLAST截取过的长度为81bp的序列），data2修改为之前保存的  K_tupledata.txt文件路径，修改完毕后运行，会得到test_data.mat，保存下来，这是我们需要的；
4.暂时将这里的test_data.mat与SVM-Adaboost中的train_data.mat合并，合并之前一定要编号，一定要知道哪些是你的待测试序列，因为后面我们还要拆分他们；    
5.合并之后要将他们转换成LibSVM格式的文件，执行代码是writeTolibsvm.m；
6.用fselect.py这个python脚本对合并后的数据集进行特征选择，运行后会得到.fscore、.select和.tr这三个文件，我们主要用到.fscore后缀名的文件，打开它，是一个N行2列的数据，第一列是特征的所在维度，第二列是该特征的F-score值，这里我们选择F-score>0.008的特征；
7.将经过特征选择的数据再次拆分，得到train_data.mat和test_data.mat,将这两组数据代入到SVM_Adaboost.m中并运行，输出参数output就是预测的结果：-1表示该序列不含σ54启动子；1表示该序  列含有σ54启动子。
